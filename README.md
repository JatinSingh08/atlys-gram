## **How to install and run locally ?**

```
$ git clone https://github.com/JatinSingh08/atlys-gram.git
$ cd atlys-gram
$ npm install
$ npm start
```

## **Features -**

- Like Post
- Publish post
- Emoji support in the post
- /signin & /signup pages
- Login Modal
- Protected route

## **Built with -**

- React JS
- Zustand
- React Router v6
- Tailwind css
